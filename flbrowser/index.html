About Fledger

Fledger is a decentralized resource sharing tool that runs directly in your browser. It enables secure, peer-to-peer communication without relying on central servers.

The project consists of three main modules:

  • Public Messaging Board - Share messages on a decentralized public board
  • WebTorrent Sharing - Share webpages in a torrent-like system
  • Proxy Viewing - Browse webpages through a decentralized proxy

For more information, visit web.fledg.re.
